O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão usada para criar páginas da web. Ela define a estrutura do conteúdo na web, permitindo que textos, imagens, vídeos e outros elementos sejam organizados e exibidos em navegadores.

O que são tags?

Tags são elementos fundamentais em HTML que definem a estrutura e o conteúdo de uma página web. Elas são usadas para marcar partes do texto e outros conteúdos, indicando como eles devem ser exibidos pelo navegador. Cada tag geralmente vem em pares, com uma tag de abertura e uma tag de fechamento

tags h1 até h6

As tags (h1 to h6) são usadas para definir títulos HTML. h1 define o título mais importante. h6 define o título menos importante.

tag p

Descrição: Representa um parágrafo de texto. Uso: Utilizada para agrupar e organizar blocos de texto, criando espaçamentos e uma melhor apresentação visual.

tag mark

Descrição: Destaca partes do texto que são relevantes, geralmente com um fundo amarelo. Uso: Ideal para chamar a atenção para informações importantes ou relevantes dentro de um texto.

tag small

Descrição: Apresenta texto em um tamanho menor que o texto normal. Uso: Usado para notas de rodapé, avisos legais ou qualquer informação que não precisa de destaque maior.

tags i, o, strong

i
Descrição: Formata o texto em itálico. Uso: Usado para enfatizar palavras, títulos de obras ou para indicar um termo técnico.

u
Descrição: Sublinhar texto. Uso: Pode ser usado para destacar palavras ou frases, embora seu uso deva ser cuidadoso, pois pode ser confundido com links.

strong
Descrição: Representa texto que deve ser enfatizado (normalmente em negrito). Uso: Utilizado para dar ênfase a informações importantes ou críticas dentro de um texto.

tags ol, ul, li

A tag ol é utilizada para criar listas ordenadas, onde os itens são numerados automaticamente pelo navegador. Isso é ideal quando a ordem dos elementos é relevante, como em instruções passo a passo, permitindo que o leitor siga a sequência correta. Por outro lado, a tag ul cria listas não ordenadas, onde os itens são apresentados com marcadores, sem uma sequência específica. Esse formato é útil para listar características, itens de compras ou tópicos, onde a ordem não é importante. Ambas as listas utilizam a tag li para definir cada item dentro delas. Portanto, cada elemento da lista, seja ordenada ou não, deve ser encapsulado por li. Assim, essas tags em conjunto ajudam a organizar o conteúdo de forma clara e visualmente acessível, facilitando a leitura e compreensão do usuário.

tag a

Descrição: Cria um link para outra página ou recurso. Uso: Usada para navegar entre diferentes páginas da web ou para referenciar recursos externos.

tag hr

Descrição: Insere uma linha horizontal na página. Uso: Usada para dividir visualmente seções de conteúdo, indicando uma pausa ou separação.

tags sub e sup

A tag sub é empregada para exibir texto um pouco abaixo da linha de base, sendo frequentemente usada em fórmulas químicas (por exemplo, H₂O para água) ou em notação matemática. Isso ajuda a indicar elementos que devem ser lidos de maneira diferente do texto normal. Já a tag sup formata o texto um pouco acima da linha de base, sendo comumente usada para indicar potências em expressões matemáticas (como E=mc²) ou para marcar notas de rodapé e referências bibliográficas.

tag blockquote

Descrição: Usada para citações longas, geralmente com recuo. Uso: Ideal para destacar citações de outras fontes ou textos importantes, proporcionando um contexto visual distinto.
